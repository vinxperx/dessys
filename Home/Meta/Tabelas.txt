====== Tabelas ======
Criado segunda 30 maio 2022

#tabela# - Descreve uma tabela
#campo# - Descreve um campo dentro de uma tabela
#int# - integer
#varchar# - normalmente varchar com máx de caracteres
#datetime# - data e hora
#char#x# - caracteres
#aut# - automático
#PK# - chave estrangeira de outra tabela
#chaveincompleta# - chave para um status ou outra entidade que não tem uma tabela de referência no sistema
#letramorta# - um campo que existe apenas pro-forma, como requício de um fluxo antigo.
#naoselect# - um código de campo que não faz referência a uma tabela, mas que está provavelmente no código
